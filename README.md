# ТВЭРС

ТВЭРС — это тестовая WordPress-тема, посвященная лампам ИКЕА. В ней есть нейросетевые статьи, поиск, рубрики и теги, AJAX-подгрузка материалов. Код написан с использованием SCSS и Laravel.
